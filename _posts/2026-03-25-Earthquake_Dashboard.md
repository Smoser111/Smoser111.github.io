This Tableau dashboard visualizes global earthquake activity over a 30-day period, providing an interactive and time-driven analysis of seismic events. All visualizations are synchronized through a centralized date filter, allowing users to dynamically explore how earthquake patterns change on a day-by-day basis.

The dashboard features a geospatial map that plots all recorded earthquakes for the selected date, enabling quick identification of geographic distribution and regional clustering. This allows users to observe where seismic activity is most concentrated at any given point in time.

To highlight significant events, the dashboard includes a “Top 10 Largest Earthquakes” view, which updates based on the selected date. This helps users quickly identify the most impactful earthquakes occurring within the chosen timeframe. In addition, a percentage breakdown of earthquakes by location provides a comparative view of how seismic activity is distributed globally on that day.

The lower section of the dashboard focuses on key analytical trends, including earthquake frequency, average magnitude, and maximum magnitude. These visualizations provide insight into both the volume and intensity of earthquakes, helping users understand how seismic activity fluctuates over time.

Overall, this dashboard demonstrates the use of Tableau for interactive data exploration, combining geospatial visualization with temporal filtering to uncover meaningful patterns in earthquake data. It highlights the ability to design cohesive, user-driven dashboards that transform raw data into actionable insights.
